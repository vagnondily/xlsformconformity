# Audit Interactif d'XLSForm (Shiny R)

Ce projet fournit une application Shiny interactive développée en R pour auditer automatiquement les erreurs courantes dans les fichiers XLSForm (utilisés par ODK, KoboToolbox, etc.).

## Prérequis et Utilisation

Vous devez avoir R et RStudio installés. Les packages R suivants sont nécessaires :

```R
install.packages(c("shiny", "readxl", "dplyr", "stringr", "openxlsx"))

## Pour lancer l'application, ouvrez app.R dans RStudio et cliquez sur le bouton "Run App", ou exécutez dans la console :
```R
shiny::runApp()
